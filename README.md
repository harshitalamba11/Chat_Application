## 🚀 Features

- 💬 **Real-time chat** with Socket.IO
- 🔔 **Typing feedback indicator** (e.g., "User is typing…")
- 📢 **Sound notifications** on new messages
- 🧑 **Live client count** display
- 📅 **Timestamps** using Moment.js
- 🔁 **Auto-scroll** to latest messages
